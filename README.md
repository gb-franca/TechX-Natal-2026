# TechX Natal 26 🚀

Este é o site oficial do evento **TechX Natal 26**, com o tema *"Data Science Interdisciplinary in Real Life"*. O projeto foi desenvolvido com foco em performance, design moderno e animações fluidas para proporcionar a melhor experiência aos participantes.

## 📍 Sobre o Evento
O evento acontecerá nos dias **15 e 16 de Maio de 2026** na **UFRN — Natal, RN**. É uma realização do IEEE CS .
